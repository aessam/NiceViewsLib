Component Library

Enum-driven sizing, style variants, optional callbacks, and @ViewBuilder generics — consistent patterns across all 27+ components.

MediaCard

Album/song cards with small, medium, large sizing via MediaCardSize enum. Rounded corners, image overlay, and subtitle support.

ActionButton

Primary, secondary, and text button styles. Supports icon + title combos with optional action callbacks.

StatusBadge

LIVE, NEW, PRO, TRENDING badges with color-coded styles via BadgeStyle enum.

UserAvatar

Profile pictures in small, medium, and large sizes via AvatarSize enum. Initials fallback with customizable colors.

ListItem

Generic list rows with image, title, subtitle. Supports selection states, loading indicators, and trailing accessories.

HeroSection

Featured content banner with background image, badge overlay, title, subtitle, and action buttons.

SearchBar

iOS-native search input with placeholder text and binding. Adaptive theming for light/dark mode.

BottomTabBar

Custom tab navigation with SF Symbols, selection state, and AnyView type erasure for dynamic destinations.

IconButton

Compact icon-only button with SF Symbol support, ideal for toolbars and inline actions.

SectionHeader

Section title with optional "See All" trailing action. Used across Home, Explore, and Library views.

Media Card Grid

Responsive grid layout showing multiple MediaCards in a LazyVGrid arrangement.

ListItem Variations

Multiple list item configurations: with subtitles, icons, trailing info, and different content layouts.

Architecture & Design

Protocol-first, enum-driven, composable. Every component follows consistent patterns.

⚙

Enum-Driven Sizing

MediaCardSize, AvatarSize, ButtonStyle — all components expose their variants through Swift enums for type-safe configuration.

✔

Snapshot Testing

75+ visual regression tests using Point-Free's swift-snapshot-testing. Every component tested in both light and dark mode.

☆

Protocol-First

ComponentConfigurable protocol with associated config types. BaseViews provide generic composition foundations.

☀

Theme System

ThemeManager as @EnvironmentObject with @Published dark mode state. Automatic UITraitCollection sync.
